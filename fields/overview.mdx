title: "Content & Fields"

Keystatic fields are the building blocks of your content schema. Every collection and singleton is defined by a set of fields that determine what data editors can enter and how it is stored in your repository.

Fields are imported from `@keystatic/core` and used inside the `schema` object of a collection or singleton.

```typescript
import { collection, fields } from '@keystatic/core'

export default config({
  collections: {
    posts: collection({
      label: 'Posts',
      slugField: 'title',
      schema: {
        title: fields.slug({ name: { label: 'Title' } }),
        published: fields.date({ label: 'Published date' }),
        draft: fields.checkbox({ label: 'Draft', defaultValue: false }),
        category: fields.select({
          label: 'Category',
          options: [
            { label: 'Engineering', value: 'engineering' },
            { label: 'Design', value: 'design' },
          ],
          defaultValue: 'engineering',
        }),
        coverImage: fields.image({
          label: 'Cover image',
          directory: 'public/images/posts',
          publicPath: '/images/posts/',
        }),
        content: fields.markdoc({ label: 'Content' }),
      },
    }),
  },
})
```

## Primitive fields

Simple scalar values — strings, numbers, booleans, and dates.

<CardGroup cols={2}>
  <Card title="Text" href="/fields/text">
    Single-line or multiline string input.
  </Card>
  <Card title="Integer" href="/fields/integer">
    Whole number with optional min/max validation.
  </Card>
  <Card title="Number" href="/fields/number">
    Floating-point number with optional step and range validation.
  </Card>
  <Card title="Checkbox" href="/fields/checkbox">
    Boolean true/false toggle.
  </Card>
  <Card title="Select" href="/fields/select">
    Single option chosen from a predefined list.
  </Card>
  <Card title="Multiselect" href="/fields/multiselect">
    Zero or more options chosen from a predefined list.
  </Card>
  <Card title="Date" href="/fields/date">
    ISO 8601 date string (`YYYY-MM-DD`).
  </Card>
  <Card title="Datetime" href="/fields/datetime">
    Local datetime string from a datetime-local input.
  </Card>
  <Card title="URL" href="/fields/url">
    Sanitised URL string.
  </Card>
</CardGroup>

<CardGroup cols={2}>
  <Card title="Image" href="/fields/image">
